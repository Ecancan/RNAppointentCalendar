
# How do you contribute

##### Git Clone 
```
git clone https://github.com/Ecancan/RNAppointmentCalendar.git
```
### Install packages

##### With Yarn
```
yarn install
yarn run dev
```

##### With Npm
```
npm install
npm run dev
```

### Install Example App (With Yarn)
```
cd example
yarn install
```

##### For IOS
```
yarn run pod
yarn run ios
```

##### For Android
```
yarn run android
```
